I.Introduction

Web Technology is a collection of tools,protocols and frameworks that are used to develop and support websites.
The field of web technology is rapidly increasing with new frameworks,standards and languages.
It gives us a way to interact with hosted information,like websites.
Let us know about it briefly in the following topics.

II.Why Web Technology is more important?

 

The Web Technology is more important for the following purposes:
* Global Reach: The Internet and Web technology  allow Businesses to reach a worldwide audience with their products and services.
* Access To Data: Web Technology provides vast amount of data that can be used for analysis, insights and informed decision making.
* Innovation: Web Technology drives innovation by providing a platform for new products, services and business models.

IV.Technology with their Purpose & Version

S.No Technology Purpose Version
1. HTML HTML is a text based approach to describing how content contained within an HTML file is structured.
Web developers use HTML keywords or tags to instruct the web browser application how to format and display the content of web pages.		 HTML5
2. CSS Cascading Style Sheets is used to style and layout web pages and to
alter the font,color,size,and spacing of your content,split it into multiple columns,or add animations and other decorative features.		 CSS3
3. JAVASCRIPT Javascript is a scripting language that enables you to create a dynamically updated content,control multimedia and animate images.
It controls multimedia within web pages and allows them to become more interactive. 		ES2015
4. BOOTSTRAP Bootstrap is a free,open source front-end development framework for the creation of websites and web apps.
Designed to enable responsive development of mobile-first websites.		 Bootstrap5
5. REACT JS The React Js framework is an open source javascript framework and library developed by facebook.
It's used for building interactive user interfaces and web application on quick mode.		 React18
